
# Hami MiniMarket

Sharaxaad kooban
- Hami MiniMarket waa bog HTML/CSS/JS ah oo soo bandhiga katalog alaabo oo si responsive u shaqeeya. Isticmaalaha wuxuu ka raadin karaa magac, kala sooci karaa category, ama xadka qiimaha ku xadidi karaa.

Features
- Responsive product grid (CSS Grid)
- Raadinta magacyada alaabta (search)
- Kala-soocidda category (filter)
- Kala-soocidda qiimaha (price range)
- Hawlaha DOM-ka oo fudud oo la fahmi karo si aad u cusboonaysiin karto

Teknolojiyada la isticmaalay
- HTML5
- CSS (Grid + media queries)
- JavaScript (DOM manipulation, event listeners)

Qaab-dhismeedka faylasha (tusaale)
- index.html — markup-ka bogga
- css/styles.css — styling
- js/script.js — filtering iyo search logic
- assets/ — sawirada iyo resources kale
